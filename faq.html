Pertanyaan Umum

Temukan jawaban atas pertanyaan yang sering diajukan.

  • Kejar Paket C setara apa?

    Kejar Paket C setara dengan pendidikan formal tingkat SMA/SMK.

  • Apakah ijazah paket C bisa dibeli?

    Tidak, ijazah Paket C tidak bisa dibeli. Ijazah hanya diberikan setelah mengikuti ujian dan memenuhi syarat.

  • Dimana bisa ikut Paket C?

    Anda bisa mengikuti Paket C di lembaga pendidikan yang terakreditasi.

  • Sekolah Paket C apakah bisa online?

    Ya, beberapa lembaga menawarkan program Paket C secara online.

  • Gimana cara ikut sekolah paket C?

    Anda bisa mendaftar di lembaga penyelenggara dengan syarat yang ditentukan.

  • Apa yang dimaksud dengan pendidikan kesetaraan?

    Pendidikan kesetaraan adalah program yang setara dengan pendidikan formal.

  • Pendidikan kesetaraan apa saja?

    Program pendidikan kesetaraan meliputi Paket A, Paket B, dan Paket C.

  • Siapa saja sasaran dari program pendidikan kesetaraan?

    Sasaran program ini adalah mereka yang tidak dapat menyelesaikan pendidikan formal.

  • Apakah lulusan PKBM bisa kuliah?

    Ya, lulusan PKBM dapat melanjutkan pendidikan ke jenjang yang lebih tinggi.
